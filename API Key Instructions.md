# Guide for getting API Keys

## Gemini
- Go to https://aistudio.google.com/app/apikey
- Sign in with your google account
- Click on "Create API key" to get your API key

## Groq
- Go to https://console.groq.com/keys to login
- Click on "Create API Key" to get your API key

## Finprep
- Go to https://site.financialmodelingprep.com/ and sign up
- Click on the dashboard button at the top to get your API key

## Alphavantage
- Go to https://www.alphavantage.co/support/#api-key
- Fill up your details
- Copy your API key

## Tavily
- Go to https://app.tavily.com/ and login
- Copy your API key

## Bing search API key
- Follow this guide: https://learn.microsoft.com/en-us/bing/search-apis/bing-web-search/create-bing-search-service-resource

## AskNews (no longer available for free)
- Go to https://my.asknews.app/en/settings/api-credentials and login
- Click on "Create credentials" to create your client ID and client secret

## WolframAlpha
- Go to https://developer.wolframalpha.com/access and login
- Click on "Get App ID"
- Fill in the required details to get your API key

## OpenAI
- Go to https://platform.openai.com/settings/organization/api-keys and login to create an API key
- You will have to buy credits at https://platform.openai.com/settings/organization/billing/overview
